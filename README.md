# Job Application Tracker

a command-line tool built in Python to help track job applications during a job search.

## Features
- Add job applications with company, role, and date
- View all applications and their current status
- Update application status (Applied, Interview, Offer, Rejected)
- Data persists between sessions using JSON file storage

## Technologies Used
- Python 3.14
- JSON for data persistance
- Git for version control

## How to Run
1. Clone the repository
   git clone https://github.com/adinrobovic/job-tracker.git
2. Navigate to the folder
   cd job-tracker
3. Run the program
   python tracker.py
